## 技术架构

### Go 后端架构
- Web 框架: Gin
- WebSocket: Gorilla WebSocket
- 配置管理: Viper
- 日志: Zap
- 数据序列化: JSON

### Vue 前端架构
- 框架: Vue 3
- 构建工具: Vite
- 路由: Vue Router
- 状态管理: Pinia
- API 通信: Fetch API
- WebSocket 通信: WebSocket API
